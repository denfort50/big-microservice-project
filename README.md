# big-microservice-project

## Описание
Большой проект, реализующий микросервисную архитектуру.

## Стек используемых технологий
* Java 21
* Apache Maven 3.8.2
* Spring Boot 3.2.4
* Lombok 1.18.30
* Checkstyle 10.7.0
* Thymeleaf 3.1.2

## Требуемое окружение для запуска
* Браузер
* JDK 21
* Apache Maven 3.8
